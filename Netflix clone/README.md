# 🎬 Netflix Clone

Welcome to the **Netflix Clone**! 🍿 This is a fully functional front-end Netflix UI built using **React.js, Tailwind CSS, Firebase Authentication, and TMDB API**.

## 🚀 Features
✅ **User Authentication** (Login/Signup using Firebase) 🔐  
✅ **Browse Movies & TV Shows** 🎥  
✅ **Trending & Popular Sections** 🔥  
✅ **Movie Trailers (YouTube Embed)** 🎞️  
✅ **Add to My List** 📌  
✅ **Responsive Design** 📱💻

## 🛠️ Tech Stack
- **React.js** ⚛️
- **Tailwind CSS** 🎨
- **Firebase Authentication** 🔑
- **TMDB API** 📽️
- **React Router** 🚏
