# ✅ To-Do List Application

Welcome to the **To-Do List Application**, a simple and efficient way to manage your daily tasks! 📝✨

✅ **Add, Edit & Delete Tasks** 🗂️  
✅ **Mark Tasks as Completed** ✅  
✅ **Filter & Sort Tasks** 🔍  
✅ **Local Storage Support** 💾  
✅ **Responsive Design** 📱💻  

## 🛠️ Tech Stack

- **CSS** 🎨
- **JavaScript** ⚡
